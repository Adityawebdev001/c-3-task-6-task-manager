Concept 1

HTML Attributes vs DOM Properties

An Attribute represents the initial state defined in the HTML markup, whereas a Property is the live, dynamic value stored in the browser's DOM node.

input.getAttribute("value") Original HTML Value
input.value Original HTML Value
Type something above to see them diverge. value tracks live state, while getAttribute() stays static!

Concept 3

Browser Rendering Pipeline

How browsers turn text files (HTML, CSS, JS) into pixels on the screen. Hover over a stage to see what happens behind the scenes.

01

Bytes & Parsing

Reads raw HTML/CSS bytes off the wire and processes text characters based on encoding.

Parsing details: The network streams raw data bytes, which the browser engine decodes into text. It reads character sequences and prepares them for the tokenization engine.
02

Tokenization

Converts characters into meaningful tokens like tags, attributes, and text segments.

Tokenization details: Uses state machine parsing to generate tags (e.g. <html>, <body>) and parse attributes. These tokens serve as the structural blocks of our pages.
03

DOM / CSSOM

Builds tree models matching parent-child structures (Document Object Model & CSSOM).

Tree structures: Parses tags into Node Objects. DOM holds content structure, and CSSOM maps visual rules, cascading styles from stylesheets and media queries.
04

Render Tree

Combines DOM and CSSOM to create a layout tree, filtering out elements like display: none.

Render Tree creation: A combined model containing elements that are visible on the page. Script elements, <head>, and display: none elements are bypassed.
05

Layout & Paint

Calculates visual coordinates (geometry) and paints shapes, text, and styles to the viewport.

Layout & Painting: Reflow/Layout calculates absolute dimensions. Repaint/Painting compiles instructions for colors, borders, shadows, and text characters.
Reflow vs. Repaint: Modifying layout properties (like width or position) triggers recalculating layout geometry (Reflow) + painting. Modifying visual properties (like background-color) only triggers painting.
